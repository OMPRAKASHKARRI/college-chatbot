# 🤖 College Chatbot

An AI-powered chatbot designed to assist students with college-related queries such as courses, schedules, events, and general information.

## 🚀 Features

* 💬 Real-time chatbot interaction
* 🎓 Answers college-related queries
* ⚡ Fast and responsive UI
* 🔗 API integration (AI-based responses)
* 🧠 Smart query handling

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript / React
* Backend: Node.js, Express
* Database: MongoDB (if used)
* AI API: OpenAI / Groq (mention what you used)
* Deployment: Vercel / Render

## 📂 Project Structure

```
client/      → Frontend code

```

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

2. Install dependencies

```
npm install
```

3. Setup environment variables
   Create a `.env` file in the server folder and add:

```
GROQ_API_KEY=your_api_key
MONGO_URI=your_mongodb_url
```

4. Run the project

```
npm start
```
